TokoVOIP
TokoVOIP is our in-game VOIP replacement, using a custom made TS Plugin.

It give us a lot more possibilities, which currently allow such things as:
  • Better voice quality
  • Radio system
  • Proximity chat unique to each player
  • Phone calls
  • Radio clicks
  • Radio effects

TokoVOIP has to be installed and running to play on our servers.

If you cannot join the appropriate server channel, please join channel in teamspeak, so that a support team member can help you.

You can find more information and download links at: http://forums.rmog.us
